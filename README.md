# 🏬 Shopping Mall Management System

A web-based application built using **Python** and **Flask** to simplify the management of shopping mall operations like store handling, inventory control, sales tracking, and customer information.

---

## 📌 Overview

The Shopping Mall Management System is designed to streamline operations within a shopping mall by allowing admins and managers to:

- Add, update, or delete store information  
- Manage product inventory  
- Process and record sales transactions  
- Maintain customer profiles and loyalty data  
- Ensure secure, role-based access

---

## 🧰 Technologies Used

- **Backend**: Python, Flask  
- **Frontend**: HTML, CSS (Jinja templating)  
- **Database**: In-memory (can be extended to SQLite/MySQL)  
- **Testing**: unittest

---

## 🚀 How to Run the Project

### ✅ Requirements

- Python 3.6 or higher  
- Flask

### 🛠 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Akshai-krishna-2003/Shopping-Mall-Management-System.git
   cd Shopping-Mall-Management-System

2. Install dependencies

3. Run the app:
   python app.py

4. Open your browser and go to: http://localhost:5000

## 🔐 Features
- 🔒 Admin and store manager login system
- 🏪 Store creation, editing, and removal
- 📦 Inventory management per store
- 💰 Sales transactions with receipts
- 👥 Customer data management
- 📊 View sales and stock history
